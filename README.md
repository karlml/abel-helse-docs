# Abel Helse - Kundereise og Sammenligning

En omfattende dokumentasjon om Abel Helses kundereise, tjenesteinnhold og sammenligning med andre helseveiledere i Norden og internasjonalt.

## Innhold

- Introduksjon til Abel Helse
- Kundereise for ulike målgrupper (bedrifter, privatpersoner, NAV/lege-henviste)
- Tjenesteinnhold, metoder og verktøy
- Veiledernes kompetanse
- Effektmåling og dokumenterte resultater
- Sammenligning med nordiske og internasjonale aktører

## Teknologi

Ren HTML/CSS single-page website med responsivt design.

## Visning

Åpne `index.html` i nettleseren eller besøk GitHub Pages URL-en når repoet er publisert.
